# Yeomyeong Wep page
> [Web] 여명종합건설사 홈페이지 제작
<br/>

## 📖 Introduction  
워드프레스를 이용해 건설회사의 대표페이지 제작을 맡게되었습니다.
  <br/><br/><br/>
## 👨‍💻 System requirements
기본적으로 워드프레스를 이용해 페이지 전체적인 틀을 구성하고,<br/>
elementor를 이용해 세부 디자인을 변경했습니다.
호스팅은 bluehost를 이용했습니다.
  <br/><br/><br/>
